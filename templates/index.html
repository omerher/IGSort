More Information

How it Works We use Instagram's API to get accurate and recent information quickly. The data is gathered in seconds, with an average time of 3 seconds per 100 posts.
How to Use To begin, simply enter an Instagram username and the number of posts you want sorted, then relax for a couple of seconds while our code does the hard part! We gather the user's latest posts, so anything older won't be analyzed.
Controls After the posts have been loaded, use the controls to filter the type of posts or sort them with different categories. Click on the post itself to instantly go to the post on Instagram.
